This plugin pulls in web posts from Project Facet platform and generates draft posts.


== Description ==

Project Facet Web Posts is a bridge between the collaborative editorial platform Project Facet.

On Facet, users in newsrooms can create various versions of a story as needed for various media platforms.

This plugin provides the ability to push the webfacet from the Facet platform to a connected
WordPress website and create a draft post. From there, users can edit and layout the post as
needed before publishing the post as normal.
